### 使用方法

```
1.拥有一个windows/mac电脑
2.下载一个vscode软件
3.下载一个node.exe,并傻瓜式安装好,https://nodejs.org/download/release/v16.19.1/
4.打开cmd,输入 npm i pnpm -g
5.将项目克隆到本地,或者下载到本地,用vscode打开
6.打开vscode终端,输入pnpm i安装依赖
7.安装完成后,pnpm dev可以预览
8.pnpm build打包部署
9.修改src/data目录实现定制化
```

### 安装依赖

```cmd
pnpm i
```

### 运行

```cmd
pnpm dev
```

### 构建部署

```cmd
pnpm build
```
